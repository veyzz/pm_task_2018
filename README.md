Работа программы:
1. Считываем построчно файл 'news_train.txt', создаем словарь dict с ключами-категориями, 
в значения отправляем множества, в которых содержатся слова, соответствующие этой категории.
2. Считываем построчно файл 'news_test.txt', разбиваем на слова, и проверяем каждое слово на вхождение
в определенную категорию. Категорию, куда вхождение было максимальным, считаем категорией статьи.
Результаты выводятся в файл 'news_output.txt'.

Прим.:
- Слова переводятся в нижний регистр.
- Словом считается a-zA-Zа-яА-Я, длина слова должна быть не менее 3 символов.
